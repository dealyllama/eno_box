# eno_box

`eno_box` is a self-playing "music box" project. Inspired by Brian Eno's approach to generative music—where music was created for environments like airports using tape loops and-process-driven systems—`eno_box` uses code as its primary engine to generate evolving, algorithmic soundscapes.

## Project Vision
The goal is to move away from static recordings and instead use purely programmatic logic (stochastic processes, FM synthesis, and wavetable synthesis) to create an infinite, non-repeating musical experience.
